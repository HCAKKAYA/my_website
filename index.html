NashvilleHousing – SQL Data Cleaning

This project involves a data cleaning process carried out on the NashvilleHousing dataset using SQL. In this study, date formats were standardized, missing address data was completed based on records with the same parcel, property and owner information was separated into individual columns, inconsistent values in the SoldAsVacant field (Y/N) were standardized as Yes/No, duplicate records were identified and prepared for cleaning, and unused columns were removed from the table for simplification. As a result, the dataset has become more consistent, analyzable, and ready for reporting.
